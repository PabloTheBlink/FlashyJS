Flashy.js

La librería de notificaciones más elegante y fácil de usar para JavaScript. Sin dependencias, completamente personalizable y lista para producción.

Comenzar

¿Por qué elegir Flashy.js?

Diseñado para desarrolladores que buscan simplicidad sin sacrificar funcionalidad

⚡

Ultra Ligero

Solo ~8KB minificado. Sin dependencias externas.

🎨

Personalizable

Temas, posiciones, animaciones y mucho más.

📱

Responsive

Perfecto en desktop, tablet y móvil.

Instalación

Múltiples formas de integrar Flashy.js en tu proyecto

1 Via CDN 

<script src="https://cdn.jsdelivr.net/gh/pablotheblink/FlashyJS/flashy.js"></script>
<script>
  flashy('¡Hola mundo!', 'success');
</script>

2 Via NPM 

npm install flashy-js
import flashy from 'flashy-js';
flashy('¡Hola mundo!', 'success');

3 Descarga

Descarga el archivo y úsalo localmente

Ejemplos Interactivos

Prueba diferentes tipos de notificaciones

Tipos de Notificación

// Diferentes formas de usar los tipos
flashy.success('¡Operación exitosa!');
flashy.error('Algo salió mal');
flashy.warning('Ten cuidado');
flashy.info('Información útil');

// O usando el método principal
flashy('Mensaje', 'success');
flashy('Mensaje', { type: 'error' });

Posiciones

flashy('Mensaje', { 
  position: 'top-left'    // top-left, top-center, top-right
});                       // bottom-left, bottom-center, bottom-right

Animaciones

flashy('Mensaje', { 
  animation: 'bounce'     // slide, fade, bounce, zoom
});

Temas

flashy('Mensaje', { 
  theme: 'dark'          // light, dark
});

Documentación de la API

Referencia completa de métodos y opciones

flashy(message, options)

Parámetros

message (string)

El texto de la notificación

options (object|string)

Configuración o tipo de notificación

Valor de retorno

function

Función para cerrar la notificación manualmente

Opciones de Configuración

Opción Tipo Por defecto Descripción
type string 'default' success, error, warning, info, default
position string 'top-right' top-left, top-center, top-right, bottom-left, bottom-center, bottom-right
duration number 4000 Duración en ms (0 = permanente)
closable boolean true Mostrar botón de cerrar
animation string 'slide' slide, fade, bounce, zoom
theme string 'light' light, dark
icon string null Icono personalizado (HTML/emoji)
onClick function null Callback al hacer clic
onClose function null Callback al cerrar

Métodos Disponibles

Métodos de Conveniencia

flashy.success(message, options)
flashy.error(message, options)  
flashy.warning(message, options)
flashy.info(message, options)

Configuración Global

flashy.setDefaults({
  duration: 5000,
  position: 'bottom-right'
});

flashy.getDefaults();

Control de Notificaciones

// Cerrar todas
flashy.closeAll();

// Cerrar específica
const close = flashy('Mensaje');
close();

Limpieza

// Limpiar todo
flashy.destroy();
